📋 한 번만 설정하는 방법

  1. Google Cloud Console → OAuth 2.0 클라이언트 ID 생성
  2. 애플리케이션 유형: 웹 애플리케이션 선택

  5. 내 Client ID/Secret 입력 → Blogger API v3 선택 → Authorize
  6. Exchange authorization code for tokens → Refresh token 복사
  7. 위 Refresh Token 칸에 붙여넣기 → 이제 영구 자동 갱신!
