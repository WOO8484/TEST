## ✅ 최종 결정 사항 (사용자 확인 완료)

### A. 위치
- **전부 포함**: 울산, 거제, 통영, 남해/삼천포, 남해 창선 (5곳)
- 좌표 정확히 설정 후 Open-Meteo 연동

### B. 콘텐츠 출처
- 검색 결과 + 유튜브 참고 (태공TV, 바다닥TV, 신국진어디가TV 등)
- 선사 정보: 삼천포·통영·거제 중심으로 정리 (구체적 선사명은 일반화)

### C. 기능 범위
- **낚시 일지 제외** (요청대로)
- 그 외 전부 포함:
  - 실시간 날씨 + 낚시 지수
  - 5종 어종 도감 + 모달 상세
  - 가이드북 (방법·매듭·채비·선상)
  - 인기 포인트 + 안전 수칙

### D. 디자인
- 사용자가 추천한 대로 진행:
  - 밝은 화이트 배경
  - Primary: #06b6d4 (cyan)
  - 귀여운 커스텀 SVG 로고 (웃는 한치)
  - 큰 이모지 + 부드러운 그림자 카드
  - Bottom Navigation 앱 느낌

### E. 기타
- 영어 버전: **우선 제외**
- PWA: 기본 메타 포함 (추후 v1.1에서 manifest 추가 가능)

## 🛠️ 기술적 결정

| 항목              | 결정 내용                              | 이유 |
|-------------------|----------------------------------------|------|
| 파일 구조         | 단일 `index.html`                      | GitHub Pages 최적, 배포 쉬움 |
| CSS 프레임워크    | Tailwind Play CDN                      | 빠른 개발 + 가벼움 |
| 날씨 API          | Open-Meteo (키 없음)                   | 무료·CORS 문제 없음·한국 시간대 지원 |
| 지도              | 정적 리스트 + Google Maps 링크         | Leaflet 추가 시 무거워짐 (v1.0) |
| 매듭 설명         | 단계별 텍스트 + 참고 유튜브 멘션       | 시각적 설명은 v1.1에서 SVG 애니메이션 고려 |
| 선사 정보         | 지역별 일반화 + 유튜브 채널 안내       | 실시간 예약 정보는 불가능 (정적) |
| 모달              | Tailwind + JS `<div>`                  | `<dialog>`보다 호환성 좋음 |
